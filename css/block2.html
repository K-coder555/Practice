Flexbox와 Grid 레이아웃

1. display: flex (1차원 레이아웃)

핵심 개념: 아이템을 하나의 축(가로 또는 세로)을 따라 정렬하고 분배하는 1차원 레이아웃 시스템입니다.
주요 사용처: 컴포넌트 내부의 아이템 정렬, 내비게이션 바, 버튼 그룹 등 한 방향으로 요소를 배치할 때 유용합니다.

2. display: grid (2차원 레이아웃)

핵심 개념: 아이템을 행(row)과 열(column)으로 구성된 격자에 배치하는 2차원 레이아웃 시스템입니다.
주요 사용처: 전체 페이지 레이아웃, 카드 목록, 갤러리 등 복잡한 구조를 정밀하게 제어해야 할 때 강력합니다.
