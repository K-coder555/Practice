CSS Position 속성 예제

position: static (기본값)

static은 모든 요소의 기본값입니다. HTML 문서의 일반적인 흐름에 따라 배치되며, top, right, bottom, left 속성이 아무런 영향을 주지 않습니다.
Static

position: relative

relative는 요소의 원래 위치(static일 때의 위치)를 기준으로 top, left 등으로 이동시킵니다. 이동하더라도 원래 있던 공간은 그대로 차지합니다.
Relative

position: absolute

absolute는 가장 가까운 위치 지정 조상(즉, position이 static이 아닌 부모 요소)에 대해 상대적으로 배치됩니다. 만약 그런 조상이 없다면 초기 컨테이닝 블록(보통 <body>)을 기준으로 합니다. 문서 흐름에서 완전히 제거되므로 원래 차지하던 공간이 사라집니다.

(이 회색 영역이 position: relative인 부모입니다)

Absolute

position: fixed

fixed는 뷰포트(브라우저 창)를 기준으로 위치가 고정됩니다. 페이지를 스크롤해도 항상 같은 위치에 표시됩니다. 문서 흐름에서 제거됩니다. (우측 하단 확인)
Fixed

position: sticky

sticky는 relative와 fixed의 특징을 모두 가집니다. 평소에는 일반적인 흐름(static)에 따라 배치되지만, 스크롤 위치가 설정된 임계점(예: top: 0)에 도달하면 그 위치에 고정(fixed처럼 동작)됩니다.
Sticky
