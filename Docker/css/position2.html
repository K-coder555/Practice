CSS Position 속성 (position: sticky 예제)

1. position: static (기본값)

static은 모든 요소의 기본값입니다. 문서의 일반적인 흐름에 따라 배치되며, top, right, bottom, left 속성이 적용되지 않습니다.
Static Box
(top, left 무시됨)

2. position: relative

relative는 요소가 자신의 원래 위치(static)를 기준으로 이동합니다. 이동 후에도 원래 있던 공간은 그대로 차지합니다. (점선 영역)
Relative Box
(top: 20px, left: 40px)

3. position: absolute

absolute는 문서 흐름에서 벗어나, position이 static이 아닌 가장 가까운 조상을 기준으로 배치됩니다.

3.1. Position이 지정된 부모가 있는 경우

회색 격자무늬 부모(position: relative)를 기준으로 배치됩니다.
Absolute Box 1
(top: 20px, right: 20px)

3.2. Position이 지정된 부모가 없는 경우

absolute 박스가 body 또는 초기 컨테이닝 블록을 기준으로 배치됩니다. 아래 박스는 회색 부모가 아닌, 전체 문서를 기준으로 배치된 것을 확인하세요.
Absolute Box 2
(bottom: 40px, left: 150px)

4. position: fixed

fixed는 뷰포트(브라우저 창)를 기준으로 고정됩니다. 스크롤을 해도 항상 같은 위치에 보입니다. (화면 우측 하단 확인)
Fixed Box

스크롤 테스트 영역

페이지를 스크롤하여 fixed와 sticky 요소의 동작을 확인하세요.

상단의 'sticky' 헤더와 우측 하단의 'fixed' 박스가 어떻게 동작하는지 주목해주세요.
